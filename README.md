This is a simple Todo app based on a tutorial by Coding in Flow.

The tutorial can be found at https://www.youtube.com/watch?v=siTUv1L9ymM

Instead of using Vite as a starting point, I have used Next.js.

The tutorial covers the following aspects of React:

- Creating a new React project
- Using useState
- Using useEffect
- Using props
- Using event handlers
- Using conditional rendering
- Using lists and keys
- Using local storage
- Using custom hooks
- Using TypeScript
- Using Tailwind CSS

## Getting Started

First, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
